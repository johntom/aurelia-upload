## from command prompt
1. npm install
3. au run
4. paste http://localhost:9000/ in browser 
5. install server 
## see https://github.com/johntom/trails-file-upload-example for trails backend.
6. create uploads directory in root of server
7. gulp watch
8. test upload with images
