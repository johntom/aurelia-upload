## from command prompt
1. npm install
2. au run
3. paste http://localhost:9000/ in browser 
4. install server 
## see https://github.com/johntom/trails-file-upload-example for trails backend.
5. create uploads directory in root of server
6. gulp watch
7. test upload with images
